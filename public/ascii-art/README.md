# ASCII Art Tech Stack Logos

Create images from these ASCII art designs. Use a black background with colored text matching each tech's brand color.

## Instructions
1. Copy each ASCII art below
2. Use a tool like Figma, Photoshop, or even VS Code with screenshots
3. Font: "Courier New" or "Consolas" (monospace)
4. Save as PNG with transparent or black background
5. Recommended size: 400x200px for small logos, 800x400px for larger ones
